Politique de Confidentialité

Collecte de données

Cette application accède uniquement aux données Google de l'utilisateur propriétaire pour automatiser ses tâches personnelles.

Utilisation des données

Les données sont utilisées exclusivement pour :

  • Automatisation d'emails personnels
  • Gestion automatisée du calendrier
  • Organisation de fichiers Drive

Partage des données

Aucune donnée n'est partagée avec des tiers. Usage strictement personnel.
